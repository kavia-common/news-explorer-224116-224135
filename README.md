# news-explorer-224116-224135

News Explorer is a minimalist, responsive React app to explore news by categories and keyword search, with bookmarks persisted via Supabase. It uses the Ocean Professional theme.

- Frontend: React (Create React App)
- Integrations: Newsdata.io, Supabase (optional with local fallback)
- Routing: React Router v6
- Styling: Custom CSS with CSS variables

See react_frontend/README.md for detailed setup.
